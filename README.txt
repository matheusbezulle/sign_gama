1- Abrir cmd, ir para o diretório que foi baixado o projeto do git sign_gama-master, baixar as dependências do node, executar o comando: 
	> npm install
2- Entrar na pasta client, e executar o mesmo comando, para baixar as dependencias do react:
	> cd client
	> npm install
3- Iniciar o react, ainda no diretório /client
	> npm start
4- Iniciar o node, voltando para o diretório anterior
	> cd..
	> node bin/server.js
5- Projeto iniciado, react configurado inicialmente para atender porta 3000 do localhost.

Para o banco de dados eu subi uma instância dele no RDS, então não é necessário fazer qualquer configuração para o mesmo.
